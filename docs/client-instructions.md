# Инструкция: как подключить MTProxy в Telegram

Краткая инструкция для передачи вашим клиентам (компаниям).

---

## Что такое MTProxy?

MTProxy — штатный механизм Telegram для обхода ограничений доступа. Работает **только для Telegram**, не требует VPN или отдельных приложений.

## Подключение (смартфон / компьютер)

### Вариант 1: по ссылке (самый простой)

1. Откройте ссылку, которую вы получили:
   - **На телефоне:** ссылка откроется в Telegram
   - **На компьютере:** скопируйте ссылку в браузер или откройте Telegram → вставьте в поиск

2. Нажмите **«Включить»** или **«Подключиться»**

3. Готово. Вверху чатов может появиться строка «Прокси подключен» — это нормально.

### Вариант 2: ручная настройка

**Android / iOS:**

1. Откройте **Настройки** → **Данные и хранилище** → **Использование прокси**
2. Выберите **«Добавить прокси»** → **MTProto**
3. Вставьте:
   - **Сервер:** IP-адрес из вашей ссылки
   - **Порт:** 443
   - **Секрет:** строка из ссылки (после `secret=`)

**Desktop (Windows / macOS / Linux):**

1. Откройте **Настройки** → **Подключение** → **Использовать прокси**
2. Выберите **MTProto**
3. Введите сервер, порт и секрет (как выше)

## Проверка

- Строка «Прокси подключен» вверху списка чатов — значит, всё работает
- Зайдите в любой чат — сообщения должны загружаться

## Частые вопросы

**Прокси не подключается**

- Проверьте интернет
- Убедитесь, что порт 443 не заблокирован вашим провайдером
- Попробуйте другую сеть (мобильный интернет вместо Wi‑Fi)

**Телефонные звонки не работают**

- Звонки Telegram через MTProxy недоступны, это ограничение протокола

**Можно ли удалить прокси?**

- Да: Настройки → Данные и хранилище → Использование прокси → удалите нужный прокси
